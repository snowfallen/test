git clone (HTTPS url - перший раз колм качаєш, або хочеш наробить н-кількість копій)
git pull (для отримання актуального коду - самий свежак з облака)
git add file_name (добавить файл, за яким ти хочеш, шоб слідив гіт)
git commit -m"message або короткий опис того, шо ти зробив з файлом"
git push -"добавить на удальонний репо, тобто облако"
git status - подивиться актуальну інфу по ЛОКАЛЬНИХ файлах
